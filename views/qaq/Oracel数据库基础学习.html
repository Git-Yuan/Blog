# 1.Oracle介绍

# Oracle数据库是什么？

Oracle Database，又名Oracle RDBMS，简称Oracle。是甲骨文公司推出的一款关系数据库管理系统。

Oracle数据库系统是目前世界上流行的关系数据库管理系统，拥有可移植性好、使用方便、功能强等优点，在各类大、中、小、微机环境中都适用。

Oracle是一种高效率、可靠性好的、适应高吞吐量的数据库解决方案。

# Oracle安装

安装教程 https://blog.csdn.net/pancheng1984/article/details/79823467

# 数据库和实例

Oracle数据库服务器由一个数据库和至少一个数据库实例组成。

  • 数据库是一组存储数据的文件，是一系列物理文件的集合（数据文件，控制文件，联机日志，参数文件等）。

  • 数据库实例则是管理数据库文件的内存结构，是一组Oracle后台进程/线程以及在服务器分配的共享内存区。数据库实例名用于外部链接，在我们开发的时候，jdbc连接的数据库名就是实例名。

//查看当前实例名
select instance_name from v$instance;
1
2

一个数据库可以有多个实例。

# 表空间

Oracle数据库是通过表空间来存储物理表的，一个数据库实例可以有N个表空间，一个表空间下可以有N张表。表空间(tablespace)是数据库的逻辑划分，每个数据库至少有一个表空间，一个表空间只能属于一个数据库。

创建表空间:

create tablespace receipt  
datafile 'D:\app\king\oradata\BCD\receipt.dbf'  
size 50m  
autoextend on;

2
3

# 用户

Oracle中用户的所有数据都是存放在表空间中的，很多个用户可以共用一个表空间，也可以指定一个用户只用某一个表空间。

创建用户：

create user 用户名 identified by 密码 default tablespace 表空间表;
1

给用户授权：

GRANT connect, resource TO user;
1
  • connect:只能登录数据库，对表进行CRUD的一些操作，不能进行建表等创建实体和创建数据库结构的操作。
  • resource:授予开发人员的，能在自己的方案中创建表、序列、视图等。
  • dba:拥有该角色的用户就能成为系统管理员了，它拥有所有的系统权限。
