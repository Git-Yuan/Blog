# 1.正则表达式基础知识

# 1.1 创建语法

var patt = /abc/i
/正则表达式主体/修饰符(可选)
1
2

在上面代码中，是使用正则表达字面量来创建正则对象，abc是一个正则表达式主体，主要用于检索，而后面的修饰符i指在检索期间不区分大小写。 我们除了上述的创建正则对象的方法之外，还可以利用构造函数创建正则对象。

var re = new RegExp("abc",i);
1

需要注意的一点是，使用字面量创建正则对象只需要一个\就能转义，比如想要表达,需要/\\/通过一个\来转义(/\\/表示一个斜杠\ )，而使用构造函数创建正则对象，需要new RegExp("\\\\"),需要四个斜杠才能代表一个斜杠。

# 1.2 修饰符

在上文中，我们用到了修饰符，修饰符的作用主要是在匹配当中起到限制作用，修饰符有以下三种，且可随意搭配使用，不用区分前后顺序。

修饰符 作用
i 忽略大小写
g 全局匹配，匹配到最后一个值为止
m 多行匹配，遇到换行不停止，直到结束

# 1.3 正则相关方法

search()方法
检索字符串中是否有指定的正则表达式的子字符串，有则返回该字符串的首个位置。

var str = "Visit Runoob!"; 
var n = str.search(/Runoob/i);  //n=6
1
2

replace()方法
使用一些字符串去代替另一些字符串

var str = "aaaannbdasasdasfasf";
var txt = str.replace("nn","hello");
var txt = str.replace(/\n/i,"hello");
1
2
3

test()方法
test()是一个正则表达式方法，用于检测一个字符串中是否含有匹配的文本，存在则返回true，不存在则返回false。

var patt = /e/;
patt.test("The best things in life are free!");  //true
1
2

exec()方法
exec也是一个正则表达式方法，匹配字符串中是否含有正则表达式的匹配，存在则返回一个数组，数组中存放匹配的结果，不存在则返回null。

    var rex = /aaa/g;
    var arr = rex.exec("The aaaaaa things aaa laaaare free!"); //['aaa']
1
2

# 1.4 方括号

方括号用于查找某个范围内的字符

表达式 作用
[abc] 查找方括号内的任何字符
[^abc] 查找任何不在方括号之间的字符
[a-z] 查找任何小写的a到z的字符
[0-9] 查找任何0-9的数字
[A-Z] 查找任何大写的A到Z的字符
[A-z] 查找任何大写A到小写z的字符

# 1.5 元字符

带有特殊含义的字符

表达式 作用
. 查找单个字符，除了换行和行结束符
\w 查找字母字符
\W 查找非字母字符
\d 查找数字
\D 查找非数字
\s 查找空白字符
\S 查找非空白字符

以上介绍的是最基础的正则表达式的知识，就不在这里一一赘述了，想要了解更多精彩请前往 https://www.runoob.com/jsref/jsref-obj-regexp.html

# 1.正则表达式进阶知识

学会了上面的基础知识，接下啦要学点进阶的正则表达式，相比较而言也会更难一点。

# 1.1 贪婪匹配与非贪婪匹配

在默认设置下，我们的正则匹配的量词都是贪婪匹配，就是尽可能多的匹配字符，比如一个"The aaaaaa things aaa laaaare free!",我们用/.+\s/去匹配以空格结尾的字符串。

    var rex = /.+\s/g;
    var arr = rex.exec("The aaaaaa things aaa laaaare free!"); //The aaaaaa things aaa laaaare
1
2

默认情况下，会匹配到最后一个空格的字符串才会结束。 而我们在量词的后面加一个？，就会结束贪婪模式，实现非贪婪模式

    var rex = /.+?\s/g;
    var arr = rex.exec("The aaaaaa things aaa laaaare free!"); //The
1
2

# 1.2 分组模式

分组就是在正则表达式中使用()括号包裹起来，包裹起来则表示一个分组。

比如我们想匹配hahaha这个字符串，我们的表达式可以是：/hahaha/，但也可以用另一种写法：/(ha){3}/，这里的(ha)就是一个分组。

# 候选

在一个分组中，可以有多个候选表达式，用|分割。

    var rex = /(a|b|c)/;
    var a = rex.test("aaaaa"); //true
    var b = rex.test("bbbbb"); //true
    var c = rex.test("ccccc"); //true
1
2
3
4

# 捕获和引用

在一个正则表达式中可以有多个分组，当分组多了之后，我们要如何获取到被捕获的每个分组的值呢？

    var str = "2020/08/26";
    var rex = /(\d{4})\/(\d{2})\/(\d{2})/;
    rex.test(str);
    console.log(RegExp.$1,RegExp.$2,RegExp.$3); //2020 08 26
1
2
3
4

通过上述代码，我们可以了解到每个分组被捕获之后，会从最开始编号$1，$2，$3，这样我们就能很好的获取到被捕获的值。

而通过与replace方法的配合，我们可以解决一些问题。比如我们想把2020/08/26改为2020-08-26

    var str = "2020/08/26";
    var rex = /(\d{4})\/(\d{2})\/(\d{2})/;
    str = str.replace(rex,'$1-$2-$3');  //2020-08-26
1
2
3

# 反向引用

被捕获的值在正则表达式中也是可以引用，这被称为反向引用。

var reg = /(\w{3}) is \1/
 
reg.test('kid is kid') // true
reg.test('dik is dik') // true
reg.test('kid is dik') // false
reg.test('dik is kid') // false
1
2
3
4
5
6

\1就是第一个被捕获的值，但如果编号超出了边界，则会被当成普通的表达式。

var reg = /(\w{3}) is \6/;
reg.test( 'kid is kid' ); // false
reg.test( 'kid is \6' );  // true

# 非捕获型分组

非捕获型分组，顾名思义，就是该分组不会被捕获，也就不会被编号，所以使用$是无法获取到值的。

var reg = /(?:\d{4})(\d{2})(\d{2})/
var date = '20200826'
reg.test(date)
 
RegExp.$1 // 08
RegExp.$2 // 26

# 正向，反向零宽断言分组

看名字可能啥都看不懂，但是上两个例子就全都明白了。

  • 正向断言分组
var reg = /kid is a (?=doubi)/
reg.test('kid is a doubi') // true
reg.test('kid is a shabi') // false
1
2
3

由上述代码可以得知，正向断言分组就是在kid is a 后面的值是否是分组中的值，如果是则返回true，不是则返回false。

  • 正向否定断言分组
var reg = /kid is a (?!doubi)/
reg.test('kid is a doubi') //false
reg.test('kid is a shabi') //true
1
2
3

正向否定断言分组就跟那正向相反。

  • 反向断言分组
    var reg = /(?<=#)\d+/;
    reg.test('#153232323'); //true
1
2
  • 反向否定断言分组
    var reg = /(?<!#)15323/;
    var str = reg.test('#153232323'); //false
